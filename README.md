# FinalProject (경영정보처리론)
# 합격했는데, 왜 떠나는가?

청년 첫 일자리 조기이탈의 규모, 원인과 이탈 이후의 이동을
데이터로 분석한 Streamlit 대시보드입니다.

## 1. 연구 질문

1. 얼마나, 언제, 누가 이탈하는가?
2. 왜 이탈하는가?
3. 조기이탈은 정말 실패인가?

## 2. 사용 데이터

- 한국노동패널조사 개인 자료
- 한국노동패널조사 직업력 자료
- 경제활동인구조사 청년층 부가조사
- 근로환경조사

## 3. 주요 분석

- Kaplan-Meier 생존분석
- 퇴직 사유 빈도 및 연도별 구성비 분석
- 로지스틱 회귀분석
- 조기이탈 이후 재취업률 분석
- 첫 직장과 다음 일자리 질 비교
